一、自带快捷使用教程

以下命令均在自己本地的Blog文件目录下右键Git Bash Here弹出的命令框中使用。

创建新文章

hexo new "New Blog Name"

简写： hexo n "New Blog Name"

官网更多内容>>>

开启服务

hexo server

简写:hexo s

开启后，可以使用localhost:4000进入浏览器中查看本地服务是否启动成功。

官网更多内容>>>

静态页面生成

hexo generate

简写：hexo g

生成静态页面，在本地查看服务前应先生成页面。

官网更多内容>>>

发布新内容到网站

hexo deploy

简写：hexo d

每次修改MD文件内容后，先使用hexo g生成静态页面，再发布该页面，发布后内容将自动同步到github。

Tips：刚发布内容，可能不会直接刷新Blog的页面，多刷新几次或稍晚再刷新查看
